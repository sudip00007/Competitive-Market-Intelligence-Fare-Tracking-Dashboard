# ✈️ Competitive Market Intelligence & Fare Tracking Dashboard

## 📌 Project Overview

The **Competitive Market Intelligence & Fare Tracking Dashboard** is a data analytics project that analyzes airline fare data to uncover pricing trends, customer demand patterns, and competitive market insights. The project demonstrates an end-to-end analytics pipeline including **data extraction, cleaning, transformation, exploratory analysis, and dashboard visualization**.

The dashboard helps airlines, revenue analysts, and business analysts understand how flight prices vary based on departure time, travel day, airline, and other market factors.

---

## 🎯 Objectives

* Analyze airline pricing behavior.
* Track fare variations across airlines.
* Identify high-demand travel periods.
* Discover time-of-day pricing premiums.
* Generate visual insights for revenue management and competitive analysis.

## 📊 Workflow

### 1. Data Extraction

* Imported airline fare dataset from Excel.
* Loaded data into Pandas DataFrame.

### 2. Data Cleaning & Transformation

Performed preprocessing including:

* Date conversion
* Time conversion
* Missing value handling
* Feature engineering
* Weekday extraction
* Departure hour categorization
* Time bucket creation

### 3. Exploratory Data Analysis

Analyzed:

* Average ticket prices
* Airline-wise fare comparison
* Weekend vs weekday pricing
* Departure time pricing trends
* Route-level fare distribution
* Market pricing behavior

### 4. Dashboard Visualization

Generated multiple business-focused visualizations to support pricing strategy and competitive intelligence.
